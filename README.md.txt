# AmiiboShop

Hola! Este repositorio es una tienda de amiibos creada utilizando la api AmiiboApi, puedes acceder a ella con el siguiente link: https://www.amiiboapi.com/docs/
Para utlizar este repositorios:
 1. Clonarlo en tu editor de texto favorito
 2. Instalar las dependencias utilizando "npm i" o "yarn add" dependiendo del gestor de paquetes que utilices
 3. Iniciar el proyecto con el comando "react-native start" y esperar unos pocos segundos.
 4. Ahora tienes que utilizar el comando "react-native run-ios" si tu dispositivo es IOS, en caso de que sea android utliza el siguiente comando "react-native run-android"
 5. Listo! Tu proyecto deberia estar corriendo sin problemas en tu dispositivo.

# Notas:
En caso de que estes utilizando android puede que tengas que hacer algunas configuraciones a tu telefono para evitar inconvenientes:
 1. Activar el modo desarrollador
 2. Activar la depuracion de usb
 3. Instalar via USB
